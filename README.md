## Quick Start

### Release Archives
You can download the latest release from the [releases page](https://github.com/jwkunz/Sabio/releases).

### Building from Source
To build Sabio from source, follow these steps:
1. Clone the repository: `git clone https://github.com/jwkunz/Sabio.git`
2. Navigate into the cloned directory: `cd Sabio`
3. Install dependencies: `npm install`
4. Build the project: `npm run build`

## Prerequisites
- Node.js (>= 14.x)
- npm (>= 6.x)
- Git

## npm Scripts
| Script           | Description              |
|------------------|--------------------------|
| `npm run build`  | Builds the project       |
| `npm start`      | Starts the application   |

## Requirements
- Ensure that your system meets the prerequisites for a successful build.

## Platform-specific Builds

### Linux
- To install on Linux, use the following commands:
  ```bash
  sudo apt update
  sudo apt install nodejs npm
  npm install -g sabio
  ```

### macOS
- To install on macOS, you can use Homebrew:
  ```bash
  brew install node
  npm install -g sabio
  ```

### Windows
- For Windows, use the following steps:
  1. Download and install Node.js from [nodejs.org](https://nodejs.org/)
  2. Open Command Prompt and run:
  ```cmd
  npm install -g sabio
  ```
